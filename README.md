# Electric vehicle Charging Analysis & Forecasting

## Overview
Analyzed 15K+ electric vehicle (EV) charging session data to uncover demand and revenue trends and built short-term forecasts to support operational planning and pricing decisions.

## Tools & Technologies
- Python (Pandas, NumPy)
- Machine Learning (Scikit-Learn)
- SQL
- Power BI

## Key Work Done
- Cleaned and transformed EV charging session data using Python  
- Designed a star schema with fact and dimension tables  
- Performed SQL-based analysis on energy usage and revenue trends  
- Built machine learning forecasts for monthly energy consumption and charging revenue  
- Developed interactive Power BI dashboards to monitor KPIs and forecast performance  

## Key Metrics
- Total Energy Consumed (kWh)
- Total Charging Revenue (USD)

## Outcome
- Identified growth trends in EV charging demand  
- Forecasted next 3 months of energy consumption and revenue  
- Created dashboards to support data-driven planning and decision-making
